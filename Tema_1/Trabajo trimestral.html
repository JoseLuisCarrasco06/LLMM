<div> Etiqueta

La etiqueta <div> es un elemento de bloque utilizado para agrupar otros elementos HTML.

Ejemplo de Código:

<div>
    <h1>Este es un encabezado</h1>
    <p>Este es un párrafo dentro de un elemento div.</p>
</div>

Caso de Uso:

Usa la etiqueta <div> para agrupar contenido, como un encabezado, navegación y sección de contenido en una página web.
